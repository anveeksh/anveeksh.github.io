📌 Your current blog platform is Wix (anveekshmrao.com/blog). You can either keep using Wix to write posts (recommended for simplicity) or migrate to a self-hosted option like Ghost, Hashnode, or Dev.to. Both paths are explained below.
OPTION A — Keep Wix Blog (Easiest)
# Steps to write & publish a new post on your existing Wix blog:
1
Log in to Wix
Go to manage.wix.com → log in with your account.
2
Open Blog Manager
In the dashboard left sidebar → click Blog → Posts → + New Post.
3
Write Your Post
Use Wix's rich text editor. Add a cover image, write your content, add tags (e.g. cybersecurity, pentest, IoT), and set the SEO title + meta description.
4
Publish
Click Publish — it goes live at anveekshmrao.com/blog/your-post-slug immediately.
5
Update This Page
After publishing on Wix, ask me to add the new post card to this page with its URL and excerpt. I'll update the HTML for you.
OPTION B — Publish This New Site (GitHub Pages)
# Free hosting — your site will be live at username.github.io
1
Create a GitHub Account
Go to github.com → sign up (free). Username suggestion: ishrao or anveekshmrao.
2
Create a Repository
Click + New Repository → name it exactly yourusername.github.io → set to Public → click Create.
3
Upload Your Files
Click uploading an existing file → drag in all 4 HTML files (index.html, projects.html, certifications.html, blog.html) + the css/ and js/ folders → Commit changes.
4
Enable Pages
Go to repo Settings → Pages → Source: Deploy from branch → Branch: main → Save. Wait ~2 minutes.
5
Your Site is Live!
Visit https://yourusername.github.io — share this link. You can also connect your custom domain anveekshmrao.com by pointing DNS to GitHub in Settings → Pages → Custom domain.
OPTION C — Netlify (Drag & Drop, 30 seconds)
# Easiest way to get it live right now:
1
Go to Netlify Drop
Visit app.netlify.com/drop — no account needed initially.
2
Zip your folder
Put all site files (index.html, projects.html, certifications.html, blog.html, css/, js/) into one folder → right-click → Compress/Zip.
3
Drag the folder onto Netlify
Drag the folder (not the zip) onto the Netlify Drop zone. Your site is live in ~10 seconds at a random URL like amazing-name-123.netlify.app.
4
Connect Custom Domain
Sign up for free → Site settings → Domain Management → Add anveekshmrao.com. Update your DNS records at your domain registrar to point to Netlify.
